📝 Sistema de Gestión de Tareas (PFO2)

Este proyecto implementa una API REST en Python usando Flask, con autenticación básica y persistencia en SQLite.

🚀 Cómo ejecutar el proyecto

  1. Cloná este repositorio o descargá los archivos.
  2. Instalá dependencias: 
    pip install flask werkzeug
  3. Ejecutá el servidor: 
    python server.py

📡 Endpoints disponibles

  • POST /registro: Registra un usuario.
  • POST /login: Inicia sesión.
  • GET /tareas: Muestra una bienvenida en HTML.

📚 Respuestas Conceptuales

❓ ¿Por qué hashear contraseñas?

Porque guardar contraseñas en texto plano es inseguro. Hashear significa cifrarlas de manera irreversible, para que ni el sistema ni atacantes puedan leerlas directamente.

❓ Ventajas de usar SQLite
  • Liviana y sin servidor adicional.
  • Ideal para pruebas o aplicaciones pequeñas.
  • Fácil de usar desde Python sin instalación externa.

🧪 Pruebas con curl

# Registro
curl -X POST http://localhost:5000/registro -H "Content-Type: application/json" -d "{\"usuario\":\"seba\",\"contraseña\":\"1234\"}"

# Login
curl -X POST http://localhost:5000/login -H "Content-Type: application/json" -d "{\"usuario\":\"seba\",\"contraseña\":\"1234\"}"

# Ver bienvenida
curl http://localhost:5000/tareas
